Agulhas de Tricô

Existem vários tipos de agulhas: retas, circulares e de ponta dupla, cada uma com seu propósito específico.

  • Agulhas retas para projetos planos
  • Agulhas circulares para peças tubulares
  • Agulhas de ponta dupla para pequenos projetos circulares

Fios e Lãs

A escolha do fio é fundamental para o sucesso do seu projeto. Cada tipo tem características específicas.

  • Lã natural para peças quentes
  • Algodão para peças de verão
  • Misturas sintéticas para durabilidade
